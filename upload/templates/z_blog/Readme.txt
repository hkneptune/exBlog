####模板介绍####
Z-Blog 是个最近经常更新的日志程序，至少在 Blueidea 经常看到相关信息。
呵呵！
通过 ASP 架构的，没用过，但看起来不错。
咱们还是说 模板 吧！
它是通过程序生成标准的 XHTML 页面代码，而它所谓的模板就是一个 CSS+图片 的包。
哇哦哦哦哦！
很酷的！能够保证它的页面代码通过标准！
太酷了！
当然了！更加方便我的移植喽！简直是一劳永逸！
通过这个模板，使用 exBlog 的朋友可以通用 Z-Blog 的模板！
Z-Blog界面样式下载： http://www.rainbowsoft.org/zblog/post/139.asp
####模板使用####
步骤：
 + 下载 Z-Blog界面样式包
 + 解压缩，放置到 exBlog 的 z_blog 模板下的 style 目录之中
 + 看下面的提示修改相应文件
 + 修改 exBlog 的 z_blog 模板下的 index.html 样式链接处
 + 登陆 exBlog 后台选择使用 z_blog 模板
 + 看看效果吧
================
这个还是很重要的！
由于 Z-Blog 编码使用了 UTF-8 它的 css 文件也转成了 UTF-8编码。
当你直接使用的时候，会发现中文成为了 乱码 ！
解决方法很简单，打开其 *.css 文件，寻找所有 font-family 属性，将其中的中文字体删掉，就解决了！
