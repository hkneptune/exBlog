    • 功能介绍 返上

    + 使用了模版(templates)技术，能让网站在瞬间改变样式；
    + 本程式副带5套模版(可自已制作模片，无须接触任何PHP代码)；
    + 用户分级管理，不同等级有不同管理权限；
    + 支持首页调用，能自由调用各栏目或所有栏目最新文章；
    + 支持公告，日历功能(能通过日历选择当前发表的BLOG记录)；
    + 支持用户论评功能，可对BLOG文章自由进行评论；
    + 添加&删除&编辑[BLOG,友情链接,公告,用户]；
    + 支持WAP浏览以及WAP发表、编辑、管理；
    + 隐藏日志功能；
    + 分类，链接重新排序功能；
    + 站长回复评论功能；
    + 修改点添加/编辑公告, 提示语法错误, 屏幕左下脚有黄色图标表示出错
    + 所有标签都应类似[url]支持选中对象后自动用标签夹住
    + 修改 i7c模板的RSS2.0在浏览器中点击白屏
    + 解决了优化、备份数据库是对整个库操作, 有安全性隐患
    + 解决了非图片文件, 如exe等其他文件类型无法上传
    + 更多...
    安装说明 返上
    1. 系统要求
    　　===========================================
    　　正常使用该软件您必须保证您的
    　　服务器满足如下要求：
    　　1) 软件环境：Apache 1.3.X + PHP 4.2.0 及以上
    　　2) 数 据 库：MySQL 3.23.X 及以上
    　　3) 其 他：服务器必须安装 微软IE浏览器4.0版本以上，推荐IE5.5+
    [本程序编写环境] EasyPHP 1.6.0.0 ( 偶懒,没配置环境 ~!~ )

    2. 系统安装
    　　===========================================
    　　1)在支持PHP的服务器上解开本软件包。
    　　2)将 ./upload 目录下所有文件上传至服务器。首页面地址是：index.php
    　　3)登录MySQL，建立一个新的数据库，命名任意，如 exblog。
    　　BTW:如果你无权限建表,请在config.inc.php中添写ICP给的数据库名
    　　4)将 ./install/config.inc.php 中相关内容修改为您的信息。
    　　注意：一定要将数据库的名称改为你刚才新建的名称，如上应为 exblog
    　　5)在IE地址栏运行 <安装路径>/admin/install.php 按提示进行建表，创建管理员等步骤。
    　　6)安装完成! 在IE地址栏运行 <安装路径>/admin/login.php 登录后台管理程式。
