    • 功能介绍 返上

    + 使用了模版(templates)技术，能让网站在瞬间改变样式；
    + 本程式副带模版(可自已制作模片，无须接触任何PHP代码)；
    + 多语言支持；
    + 针对搜索引擎优化，支持伪静态地址；
    + 用户分级管理，不同等级有不同管理权限；
    + 自由文章分类；
    + 添加&删除&编辑[BLOG,友情链接,公告,用户,论评]；
    + 丰富的UBB支持，强大的UBB文章发布后台；
    + TrackBack 支持；
    + 置顶，隐藏日志功能；
    + 允许编辑日志发布天气状况；
    + 允许编辑发布时间；
    + 分类，链接重新排序功能；
    + 站长回复评论功能；
    + 文章批量删除；
    + 可选Blog访问关闭与开放；
    + 用户登陆验证码，可选开关；
    + 优化、备份数据库；
    + 文件上传；
    + RSS 0.9x 、RSS 1.0 、RSS 2.0 和 Atom 0.3 文章聚合，最新文章、各栏目最新文章、最新评论；
    + 支持首页调用，能自由调用各栏目或所有栏目最新文章；
    + 支持公告，日历功能(能通过日历选择当前发表的BLOG记录)；
    + 文章搜索功能，并且加亮搜索关键词；
    + 支持用户论评功能，可对BLOG文章自由进行评论；
    + 友情链接前台申请；
    + 访问统计；
    + 站点键词、文章关键词等；
    + 支持WAP浏览以及WAP发表、编辑、管理；
    + QQ友好页面。可应用于QQ自定义面板的窄幅页面；
    + 更多...
    安装说明 返上
    1. 系统要求
    　　===========================================
    　　正常使用该软件您必须保证您的服务器满足如下要求：
    　　1) 软件环境：Apache 1.3.X + PHP 4.2.0 及以上
    　　2) 数 据 库：MySQL 3.23.X 及以上
    　　3) 其 他：服务器必须安装 微软IE浏览器4.0版本以上，推荐IE5.5+

    2. 如何安装
    　　===========================================
    　　本软件[exBlog]安装过程十分简便。
    　　请仔细阅读本软件[exBlog]压缩包内的 Readme.txt 或者 Install.txt 。

    3. 如何升级
    　　===========================================
    　　本软件[exBlog]升级过程同样简便。
    　　请仔细阅读本软件[exBlog]压缩包内的 Readme.txt 或者 Install.txt 。
