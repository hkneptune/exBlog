ExtensionLib::_getLatestRecords (  $  table_name,
$  num = 0,
$  order_fields = "",
$  conditions = "",
$  fields_list = "*",
$  offset = 0,
$  return_only_first = FALSE
) 
 

Get last x records from a select table.

Parameters:
$table_name Name of table
$num How many records would be fetched
$order_fields field order for sort records, include sort order. example: id desc or id desc, name asc, like ORDER in mysql
$conditions condition for fetching records. all records must be matched. like WHERE in mysql
$fields_list which fields would be return. default was all fields
$offset offset number for starting fetching records.
$return_only_first Whether return one record only.
TRUE = yes, FALSE = no. Default was no
If it was TRUE, a array which using field names as key would be return,
else a array which every element was a record in would be return
